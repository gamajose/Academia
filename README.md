# Academia Platform

Plataforma para gestao de academias com API, painel web, app mobile, software desktop e area do aluno.

## Modulos principais

- API Node.js com PostgreSQL.
- Painel web administrativo.
- Login administrativo por usuario.
- Login proprio do aluno.
- Cadastro de alunos, planos, matriculas e pagamentos.
- Check-ins.
- Treinos, fichas semanais, exercicios, videos e instrucoes.
- Analise inteligente inicial de fichas.
- Relatorios e exportacoes CSV.
- Auditoria operacional.
- Backup do banco.
- App Flutter inicial.
- Software desktop Python/Tkinter e base Electron.

## Estrutura

- apps/api: backend da plataforma.
- apps/web: painel administrativo e portal do aluno.
- apps/mobile: app Flutter.
- apps/software: software desktop Python/Tkinter.
- apps/desktop: base Electron.
- database: migrations SQL.
- scripts: automacoes e testes.
- infra: arquivos de infraestrutura.
- docs: documentacao operacional consolidada.
