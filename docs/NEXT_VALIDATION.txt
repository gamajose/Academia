Validacao das novas entregas

1. Atualizar projeto:
cd /mnt/Projetos/Academia
git pull

2. Aplicar banco e subir API:
cd /mnt/Projetos/Academia/apps/api
set -a
source .env
set +a
npm install
npm run migrate
npm run start

3. Teste automatico basico:
cd /mnt/Projetos/Academia
bash scripts/smoke-test.sh

4. Exportacoes:
Acessar http://IP_DA_VM:8084/exports.html

5. Relatorios:
Acessar http://IP_DA_VM:8084/reports.html

6. Backup manual:
cd /mnt/Projetos/Academia
set -a
source apps/api/.env
set +a
bash scripts/backup-academia.sh

7. Usuarios:
Usar os endpoints:
GET /api/users
POST /api/users
POST /api/users/activate
POST /api/users/deactivate
