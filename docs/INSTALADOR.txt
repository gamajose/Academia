Instalador e inicializacao

Arquivos criados:
apps/installer/windows-start-api.bat
apps/installer/windows-start-web.bat
apps/installer/desktop-linux.txt

Uso no Windows:
- Abrir windows-start-api.bat para iniciar a API.
- Abrir windows-start-web.bat para iniciar o painel web.

Uso no Linux:
cd /mnt/Projetos/Academia/apps/api
npm install
npm run migrate
npm run start

cd /mnt/Projetos/Academia/apps/web
python3 -m http.server 8084

Software de check-in:
cd /mnt/Projetos/Academia/apps/software
python3 academia_checkin.py
