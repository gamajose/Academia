Release v0.3.0

Objetivo:
Entregar o primeiro painel web operacional da plataforma Academia.

Entregas:
- Painel web em HTML, CSS e JavaScript.
- Login com token.
- Status da API no painel.
- Dashboard com resumo operacional.
- Cadastro de aluno.
- Cadastro de plano.
- Criacao de matricula.
- Criacao de cobranca.
- Baixa de pagamento.
- Check-in manual.
- Listas de alunos, planos, matriculas, pagamentos e check-ins recentes.
- Suporte basico a CORS na API.
- Documentacao de uso do painel.

Validacao sugerida:
- git pull
- npm run migrate
- subir API na porta 3004
- subir painel web na porta 8084
- testar fluxo completo pela tela

Tag sugerida:
git tag -a v0.3.0 -m "v0.3.0"
git push origin v0.3.0
