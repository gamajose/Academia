Auditoria operacional

Objetivo:
Registrar acoes importantes realizadas no sistema para facilitar suporte, conferencia e rastreabilidade.

Arquivos adicionados:
apps/api/lib/audit.js
apps/api/features/auditRoutes.js
apps/web/audit.js
apps/web/audit.html

Endpoint:
GET /api/audit/recent

Como acessar pela VM:
http://IP_DA_VM:8084/audit.html

Observacoes:
- A pagina usa o token salvo no navegador apos login no painel principal.
- As atividades sao carregadas automaticamente a cada poucos segundos.
- As acoes de aluno e cancelamentos ja registram eventos.
