Software Desktop - Academia Platform

Opcoes atuais:
- apps/software/academia_checkin.py: programa Python/Tkinter para recepcao, check-in, alertas e resumo de evolucao.
- apps/desktop: base Electron que abre o painel web ativo.

Funcionalidades do software Python:
- Configurar URL da API.
- Login com usuario da academia.
- Listar alunos ativos.
- Buscar aluno pelo nome.
- Registrar check-in.
- Listar ultimos check-ins.
- Ver central de alertas operacionais.
- Ver pagamentos vencidos.
- Ver fichas antigas para revisao.
- Ver resumo de avaliacao do aluno selecionado.

Como testar o software Python:
cd /mnt/Projetos/Academia/apps/software
python3 academia_checkin.py

No Windows:
python academia_checkin.py

Como testar a base Electron:
cd /mnt/Projetos/Academia/apps/desktop
npm install
npm run start

A API precisa estar ativa na porta 3004.
